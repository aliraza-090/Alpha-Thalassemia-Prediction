# Alpha-Thalassemia-Prediction
A Streamlit web app for predicting Alpha-Thalassemia carrier status from blood test parameters. Users input Hb, PCV, RBC, MCV, MCH, RDW, WBC, Neutrophils, Lymphocytes, Platelets, HbA, HbA2, and HbF. Uses a trained RandomForest model to classify as Normal or Alpha Carrier. Ideal for learning and research
